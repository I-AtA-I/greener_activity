# activity-filler
A simple bash script that lets crontab launch another script uploading a commit every 2 hours to newly created "streak" repository giving you an activity on YOUR github :)

⚠️ This project is maintained by me only.  
Feel free to clone or fork it for personal use, but I’m not accepting contributions.


STEPS:
1) Download this repo
2) Go into the activity-filler directory
3) run "activity.sh"
